# Spanish Reader — Руководство пользователя

> Версия: v1.6.0

## Введение

Spanish Reader — это расширение для браузера, созданное для изучающих испанский язык. Оно добавляет на веб-страницы и в PDF произношение и разметку слогов для испанских слов, поддерживаются акценты Латинской Америки и Испании. Также есть словарь при наведении, синтез речи и перевод — чтобы проще осваивать испанское произношение.

---

## Основные возможности

- **Режим произношения для всей страницы** — одним щелчком добавить произношение ко всем испанским словам на странице
- **Разбиение на слоги** — показ деления слова на слоги с отметкой ударения
- **Словарь при наведении** — наведите на слово, чтобы увидеть определения, произношение и кнопку динамика; режимы: Словарь, Подсказка или Отключено
- **PDF Reader** — встроенный просмотр PDF с произношением, словарём, озвучкой и переводом; drag & drop, открытие по URL и автоматическое распознавание PDF
- **Акценты Латинской Америки и Испании** — переключение между произношением es-419 (Латинская Америка) и es-ES (Испания)
- **Синтез речи** — нажмите кнопку динамика, чтобы услышать произношение в выбранном акценте
- **Озвучка выделения с эффектом караоке** — выделите текст; появится панель с кнопками озвучки и перевода; слова подсвечиваются по мере воспроизведения
- **Перевод выделения** — выделите текст и нажмите «перевести» для мгновенного перевода через Bing или Google Translate
- **Горячие клавиши** — быстрый доступ к основным функциям через настраиваемые сочетания клавиш
- **Многоязычный интерфейс** — Доступен на многих языках

## Как пользоваться

### Шаг 1: Установите расширение

Установите **Spanish Reader** из [Chrome Web Store](https://chromewebstore.google.com/) или загрузите локально в режиме разработчика.

### Шаг 2: Откройте любую страницу

Откройте любую веб-страницу с испанским текстом.

### Шаг 3: Включите произношение

Нажмите на значок расширения. Включите **Включить произношение**, затем **Режим всей страницы**, чтобы разметить все слова на странице.

### Шаг 4: Смотрите произношение

Наводите курсор на слова, чтобы увидеть подсказки с произношением. Нажимайте на значок динамика, чтобы прослушать. Слоги показывают структуру слова (например, «com·pu·ta·do·ra»).

### Шаг 5: Озвучка и перевод выделения

Выделите испанский текст мышью. Рядом появится компактная панель с двумя кнопками:
- **Читать** — произносит выделенное вслух с покадровой подсветкой слов
- **Перевести** — показывает встроенный перевод под панелью

Также можно использовать контекстное меню: **Spanish Reader > Читать выделение** или **Перевести выделение**.

> **Совет:** Откройте панель настроек по значку расширения и настройте акцент, скорость речи, режим наведения, движок перевода и другое.

## Режим произношения для всей страницы

Когда включён режим для всей страницы, над каждым испанским словом отображается произношение через ruby text.

Расширение автоматически подстраивает межстрочный интервал, чтобы подписи не наезжали на текст.

---

## Разбиение на слоги

При включённой опции **Показывать разделение на слоги** каждое слово показывается с делением на слоги:

- Слоги разделены средней точкой (·)
- Ударный слог определяется по правилам испанской орфоэпии
- Слова с письменным ударением (á, é, í, ó, ú) всегда ударяются на слог с ударной гласной

Это помогает понять:
- куда ставить ударение при произнесении
- как слова переносятся по строкам
- ритмическую структуру испанского

## Словарь при наведении

В расширении есть словарь при наведении. В настройках можно выбрать режим:

| Режим | Поведение |
|------|-----------|
| **Словарь** | При наведении: произношение + определение + кнопка динамика |
| **Подсказка** | При наведении: произношение + динамик (без определений) |
| **Отключено** | Без эффекта при наведении |

## PDF-читалка

Spanish Reader включает встроенный PDF-читалку: чтение PDF с произношением, словарём, озвучкой и переводом.

### Как открыть PDF

**Из всплывающего окна**  
Нажмите на значок расширения, затем **Открыть PDF-читалку**. Перетащите PDF или нажмите **Выбрать файл**. Можно вставить URL PDF.

**Контекстное меню**  
Щёлкните правой кнопкой по ссылке на `.pdf` и выберите **Открыть PDF с Spanish Reader**.

**Автоматическое распознавание**  
Если включено **Интеллектуальное распознавание PDF**, расширение перенаправляет URL с `.pdf` во встроенный читалку.

### Возможности PDF-читалки

- Аннотации произношения на всём тексте
- Словарь по клику на слово
- Панель выделения: озвучка, перевод или копирование
- Боковая панель: оглавление и миниатюры
- Поиск, темы (тёмная, светлая, сепия), масштаб

## Перевод

Выделите текст на странице и используйте перевод для мгновенного результата.

**Способ 1: панель выделения**  
Выделите текст и нажмите **Перевести** на панели.

**Контекстное меню** — выделите текст, щёлкните правой кнопкой и выберите **Перевести выделение**.

**Способ 3: горячая клавиша**  
Выделите текст и нажмите `Alt+Shift+T` (на Mac: `Ctrl+Shift+T`).

**Движки перевода:**
- **Bing Translate** (по умолчанию) — на базе Microsoft Translator
- **Google Translate** — на базе Google

Оба поддерживают **108 целевых языков**.

## Горячие клавиши

| Сочетание | На Mac | Действие |
|-----------|--------|----------|
| `Alt+Shift+I` | `Ctrl+Shift+I` | Включить/выключить произношение |
| `Alt+Shift+S` | `Ctrl+Shift+S` | Озвучить выделенный текст |
| `Alt+Shift+T` | `Ctrl+Shift+T` | Перевести выделенный текст |

> **Совет:** Сочетания можно изменить в Chrome: `chrome://extensions/shortcuts`.

## Частые вопросы

**В: Почему на некоторых страницах не работает?**  
О: Из соображений безопасности расширения не работают на особых страницах: `chrome://`, настройки браузера и Chrome Web Store.

**В: Нет звука при TTS?**  
О: Проверьте громкость системы и установлены ли языковые пакеты для испанского.

**В: Не работает перевод?**  
О: Нужен интернет. Если Bing Translate даёт сбой, переключитесь в настройках на Google Translate.
