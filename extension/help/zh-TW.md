# Spanish Reader - 使用手冊

> 版本：v1.5.1

## 簡介

Spanish Reader 是專為西班牙語學習者設計的瀏覽器擴充功能。它能在網頁與 PDF 上的西班牙語單字旁加入發音標註與音節切分標記，支援拉丁美洲與西班牙兩種口音。還包含離線字典、文字轉語音和翻譯功能，協助您更輕鬆地學習西班牙語發音與詞彙。

---

## 主要功能

- **整頁標註模式** — 一鍵為頁面上所有西班牙語單字加上發音標註
- **音節劃分** — 顯示單字的音節分割與重音位置（如 "com·pu·ta·do·ra"）
- **懸停字典** — 將滑鼠懸停在單字上可查看釋義、發音與喇叭按鈕；支援字典模式、提示模式和關閉
- **PDF 閱讀器** — 內建 PDF 閱讀器，支援發音標註、字典、朗讀和翻譯；支援拖放、URL 載入和 PDF 智慧偵測
- **口音切換** — 在拉丁美洲（es-419）與西班牙（es-ES）發音之間切換
- **西班牙語朗讀** — 使用 Chrome TTS 以符合口音的西班牙語語音朗讀
- **選區朗讀卡拉 OK 效果** — 選取文字後彈出工具列，朗讀時逐字高亮
- **即時翻譯** — 選取文字後透過必應或 Google 翻譯即時翻譯，結果顯示在氣泡中
- **鍵盤快速鍵** — 快速存取核心功能
- **多語言介面** — 支援多種介面語言

## 使用步驟

### 第一步：安裝擴充功能

從 [Chrome 線上應用程式商店](https://chromewebstore.google.com/) 安裝 **Spanish Reader**，或以開發人員模式在本機載入。

### 第二步：開啟網頁

造訪任何包含西班牙語內容的網頁。

### 第三步：啟用發音標註

點選瀏覽器工具列中的擴充功能圖示，開啟「啟用發音標註」，再開啟「整頁標註」模式即可為所有單字加上標註。

### 第四步：查看發音與釋義

將滑鼠懸停在單字上查看發音提示，點選喇叭圖示收聽發音。音節劃分會顯示單字結構（如 "com·pu·ta·do·ra"）。

### 第五步：朗讀和翻譯

用滑鼠選取西班牙語文字，會出現選區工具列：
- **朗讀** — 逐字高亮朗讀選取文字
- **翻譯** — 顯示翻譯結果氣泡

也可右鍵選擇「Spanish Reader > 朗讀選區」或「翻譯選區」。

> **提示：** 開啟擴充功能彈出視窗可調整口音、懸停模式、朗讀速率、翻譯引擎等設定。

---

## 整頁標註模式

開啟整頁標註後，頁面上所有西班牙語單字上方會以 ruby 樣式顯示發音標註。擴充功能自動調整行高，確保標註不與文字重疊。

---

## 音節劃分

開啟「顯示音節劃分」後，每個單字會顯示其音節結構：

- 音節之間以中間點（·）分隔
- 根據西班牙語重音規則標示重音音節
- 帶重音符號的字母（á、é、í、ó、ú）所在音節始終為重音

---

## 懸停字典

擴充功能內建**西班牙語釋義字典**（本機離線）。設定中可選擇模式：

| 模式 | 行為 |
|------|------|
| **字典** | 發音 + 西班牙語釋義 + 朗讀按鈕 |
| **提示** | 發音 + 朗讀按鈕（無釋義面板） |
| **關閉** | 無懸停效果 |

> **提示：** 字典資料僅在字典模式啟用時載入，切換模式時釋放記憶體。

---

## PDF 閱讀器

內建 PDF 閱讀器支援與網頁相同發音標註、字典、朗讀和翻譯功能。

**開啟方式：**
- 彈出視窗中點選「開啟 PDF 閱讀器」→ 拖放檔案或貼上 URL
- 右鍵 PDF 連結 →「用 Spanish Reader 開啟」
- 開啟「PDF 智慧偵測」自動重新導向 PDF

**PDF 閱讀器功能：** 整頁標註、點擊查詞、選區朗讀/翻譯/複製、側欄目錄和縮圖圖、搜尋、深色/淺色/棕褐主題、縮放

---

## 選區朗讀

**工具列** — 選取文字 → 點選朗讀按鈕，逐字高亮（卡拉 OK 效果）。

**右鍵選單** — Spanish Reader > 朗讀選區

**快速鍵** — `Alt+Shift+S`（Mac：`Ctrl+Shift+S`）

> 建議安裝系統級西班牙語語音套件以獲得最佳朗讀品質。

---

## 翻譯

**工具列** — 選取文字 → 點選翻譯按鈕，結果顯示在氣泡中。

**右鍵選單** — Spanish Reader > 翻譯選區

**快速鍵** — `Alt+Shift+T`（Mac：`Ctrl+Shift+T`）

**翻譯引擎：** 必應（預設）和 Google，支援多種目標語言。翻譯需要網路連線；僅在您觸發翻譯時傳送選取文字。

---

## 鍵盤快速鍵

| 快速鍵 | Mac | 功能 |
|--------|-----|------|
| `Alt+Shift+I` | `Ctrl+Shift+I` | 開關發音標註 |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 朗讀選取文字 |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 翻譯選取文字 |

> 可在 `chrome://extensions/shortcuts` 自訂快速鍵。

## 設定說明

| 設定 | 說明 |
|------|------|
| **啟用發音標註** | 主開關 |
| **整頁標註** | 為所有單字顯示發音 |
| **口音** | 選擇拉丁美洲或西班牙 |
| **顯示音節劃分** | 顯示音節分割標記 |
| **懸停模式** | 字典 / 提示 / 關閉 |
| **朗讀速率** | 調整句子朗讀速度 |
| **翻譯引擎** | 必應翻譯或 Google 翻譯 |
| **目標語言** | 翻譯目標語言 |
| **PDF 智慧偵測** | 自動重新導向 PDF |

補充說明：

- **首次使用：** 安裝後請開啟彈出視窗確認預設設定；已開啟的分頁可能需要重新整理。
- **效能：** 長文章整頁標註會處理更多文字；如頁面卡頓可切換為提示模式。
- **隱私：** 字典和發音標註完全在本機執行；僅翻譯功能在您主動觸發時傳送選取文字。

## 學習建議

1. **從短文開始** — 先在短新聞上啟用整頁標註，再用於長文章。
2. **結合音訊學習** — 遇到生詞點選喇叭，擴充功能會使用符合口音的語音朗讀。
3. **關注重音** — 西班牙語重音規則一致，音節劃分中標記的重音位置非常重要。
4. **PDF 學習** — 透過 PDF 閱讀器匯入課程資料，保持一致的標註體驗。
5. **口音對比** — 切換拉丁美洲和西班牙口音，聽辨發音差異（如 c/z/ll 的發音）。

## 常見問題

**問：部分頁面不生效？**  
答：擴充功能無法在 `chrome://`、瀏覽器設定頁和 Chrome 線上應用程式商店等內部頁面執行。

**問：啟用後無標註？**  
答：嘗試重新整理頁面。如果頁面在擴充功能啟用前已載入，可能需要重新整理。

**問：沒有朗讀聲音？**  
答：檢查系統音量和西班牙語語音套件。可能需要在作業系統中安裝西班牙語語音。

**問：翻譯失敗？**  
答：翻譯需要網路連線；嘗試切換翻譯引擎；企業防火牆可能阻擋 API。

**問：字典可離線使用嗎？**  
答：是的。西班牙語釋義和發音資料完全儲存在本機，無需網路。
