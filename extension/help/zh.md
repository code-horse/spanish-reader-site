# Spanish Reader - 使用手册

> 版本：v1.0.0

## 简介

Spanish Reader 是一款面向西班牙语学习者的浏览器扩展。它能在网页和 PDF 中为西班牙语单词添加发音标注和音节划分标记，支持拉丁美洲和西班牙两种口音。还包含悬停词典、文本朗读和翻译功能，帮助您更轻松地学习西班牙语发音。

---

## 主要功能

- **整页标注模式** — 一键为页面所有西班牙语单词添加发音标注
- **音节划分** — 显示单词的音节分割和重音位置
- **悬停词典** — 鼠标悬停查看词义、发音和朗读按钮
- **PDF 阅读器** — 内置 PDF 阅读器，支持发音标注、词典、朗读和翻译
- **口音切换** — 在拉丁美洲和西班牙发音之间切换
- **文本朗读** — 点击扬声器按钮收听发音
- **选区朗读卡拉 OK 效果** — 选中文本后逐词高亮朗读
- **即时翻译** — 选中文本后通过必应或谷歌翻译即时翻译
- **键盘快捷键** — 快速访问核心功能

---

## 使用步骤

### 第一步：安装扩展

从 [Chrome 网上应用店](https://chromewebstore.google.com/) 安装 **Spanish Reader**。

### 第二步：打开网页

访问任何包含西班牙语内容的网页。

### 第三步：启用发音标注

点击浏览器工具栏中的扩展图标，开启"启用发音标注"，再开启"整页标注"模式。

### 第四步：查看发音

将鼠标悬停在单词上查看发音提示，点击扬声器图标收听发音。音节划分会显示单词结构（如 "com·pu·ta·do·ra"）。

### 第五步：朗读和翻译

用鼠标选中西班牙语文本，会出现浮动工具栏：
- **🔊 朗读** — 逐词高亮朗读
- **🌐 翻译** — 显示翻译结果

## 键盘快捷键

| 快捷键 | Mac | 功能 |
|--------|-----|------|
| `Alt+Shift+I` | `Ctrl+Shift+I` | 开关发音标注 |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 朗读选中文本 |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 翻译选中文本 |

## 设置说明

| 设置 | 说明 |
|------|------|
| **启用发音标注** | 主开关 |
| **整页标注** | 为所有单词显示发音 |
| **口音** | 选择拉丁美洲或西班牙 |
| **显示音节划分** | 显示音节分割标记 |
| **悬停模式** | 词典、提示或关闭 |
| **朗读速率** | 调整朗读速度 |
| **翻译引擎** | 必应翻译或谷歌翻译 |
| **目标语言** | 翻译目标语言 |
| **PDF 智能检测** | 自动重定向 PDF |
