# Spanish Reader - 사용 가이드

> 버전: v1.5.1

## 소개

Spanish Reader는 스페인어 학습자를 위한 브라우저 확장 프로그램입니다. 웹 페이지와 PDF의 스페인어 단어에 발음 표기와 음절 구분 표시를 추가하며, 라틴 아메리카와 스페인 억양을 모두 지원합니다. 오프라인 스페인어 사전, TTS, 카라오케 읽기 및 번역 기능이 있어 스페인어 발음을 더 쉽게 익힐 수 있습니다.

---

## 주요 기능

- **전체 페이지 발음 모드** — 한 번의 클릭으로 페이지의 모든 스페인어 단어에 발음 표기 추가
- **음절 구분** — 모든 단어에 음절 나눔과 강세 표시 (예: "com·pu·ta·do·ra")
- **호버 사전** — 단어에 마우스를 올리면 뜻, 발음, 스피커 버튼 표시; 사전 / 툴팁 / 끄기 모드
- **PDF 리더** — 발음 표기, 사전, 음성, 번역이 있는 내장 PDF 리더
- **라틴 아메리카·스페인 억양** — 라틴 아메리카(es-419)와 스페인(es-ES) 발음 전환
- **스페인어 TTS** — Chrome TTS로 선택한 억양의 스페인어 음성 재생
- **선택 영역 읽기·카라오케** — 스페인어 텍스트 선택 시 도구 모음; 오디오와 함께 단어 하이라이트
- **선택 영역 번역** — Bing 또는 Google 번역; 인라인 말풍선
- **키보드 단축키** — **Alt+Shift+I** (Mac: **Ctrl+Shift+I**)
- **다국어 인터페이스**

---

## 사용 방법

### 1단계: 확장 프로그램 설치

[Chrome Web Store](https://chromewebstore.google.com/)에서 **Spanish Reader**를 설치하거나 개발자 모드로 로컬에서 로드합니다.

### 2단계: 웹 페이지 열기

스페인어 콘텐츠가 있는 웹 페이지를 방문합니다.

### 3단계: 발음 켜기

확장 프로그램 아이콘을 클릭합니다. **발음 사용**을 켠 다음 **전체 페이지 모드**를 켜 페이지의 모든 단어에 표기합니다.

### 4단계: 발음 보기

단어에 마우스를 올리면 발음 툴팁이 표시됩니다. 스피커 아이콘을 클릭하면 발음을 들을 수 있습니다. 음절 구분은 단어 구조를 보여 줍니다 (예: "com·pu·ta·do·ra").

### 5단계: 선택한 텍스트 읽기 및 번역

스페인어 텍스트를 선택합니다. **읽기** 및 **번역** 버튼이 있는 도구 모음이 나타납니다. 컨텍스트 메뉴: **Spanish Reader > 선택 영역 읽기** 또는 **선택 영역 번역**.

> **팁:** 확장 프로그램 팝업에서 억양, 호버 모드, 읽기 속도, 번역 엔진 등을 조정할 수 있습니다.

---

## 전체 페이지 발음 모드

전체 페이지 모드가 켜지면 모든 스페인어 단어 위에 ruby 스타일로 발음 표기가 표시됩니다. 줄 간격이 자동으로 조정됩니다.

---

## 음절 구분

**음절 구분 표시**가 켜지면 각 단어의 음절 나눔이 표시됩니다.

- 음절은 가운뎃점(·)으로 구분
- 강세 음절은 스페인어 강세 규칙에 따라 결정
- 악센트 기호(á, é, í, ó, ú)가 있는 단어는 항상 해당 음절에 강세

---

## 호버 사전

**스페인어 정의**가 포함된 로컬 사전 (오프라인):

| 모드 | 동작 |
|------|------|
| **사전** | 호버 시 발음 + 스페인어 정의 + 스피커 |
| **툴팁** | 호버 시 발음 + 스피커 |
| **끄기** | 호버 효과 없음 |

> **팁:** 사전 데이터는 사전 모드에서만 로드됩니다.

---

## PDF 리더

**PDF 리더 열기** ← 파일 드래그 또는 **파일 선택**, URL 붙여넣기. `.pdf` 링크 우클릭 → **Spanish Reader로 PDF 열기**. **PDF 스마트 감지**로 자동 리디렉션.

---

## 선택 영역 읽기·카라오케

텍스트 선택 → **읽기**. TTS와 함께 단어 하이라이트. 단축키: `Alt+Shift+S` (Mac: `Ctrl+Shift+S`).

---

## 번역

텍스트 선택 → **번역**. Bing(기본) 또는 Google. 단축키: `Alt+Shift+T` (Mac: `Ctrl+Shift+T`).

---

## 키보드 단축키

| 단축키 | Mac | 동작 |
|--------|-----|------|
| `Alt+Shift+I` | `Ctrl+Shift+I` | 발음 표기 켜기/끄기 |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 선택 영역 읽기 |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 선택 영역 번역 |

## 자주 묻는 질문

**질문: 일부 페이지에서 동작하지 않는 이유?**  
답변: `chrome://`, Chrome Web Store 등 특수 페이지에서는 실행할 수 없습니다.

**질문: TTS 소리가 없음?**  
답변: 시스템 음량과 스페인어 음성 팩을 확인하세요.

**질문: 사전 오프라인 사용?**  
답변: 예 — 로컬 데이터입니다.
