# Spanish Reader - 使用ガイド

> バージョン: v1.0.0

## はじめに

Spanish Reader は、スペイン語学習者向けのブラウザ拡張機能です。ウェブページや PDF のスペイン語単語に発音注釈と音節区切りを追加し、ラテンアメリカとスペインの両方のアクセントに対応しています。ホバー辞書、テキスト読み上げ、翻訳機能も搭載しています。

---

## 主な機能

- **ページ全体の発音モード** — ワンクリックで全単語に発音注釈を表示
- **音節区切り** — 各単語の音節分割とアクセント位置を表示
- **ホバー辞書** — マウスオーバーで語義・発音・再生ボタンを表示
- **PDF リーダー** — 内蔵 PDF リーダーで発音注釈・辞書・読み上げ・翻訳に対応
- **アクセント切替** — ラテンアメリカとスペインの発音を切り替え
- **テキスト読み上げ** — スピーカーボタンで発音を再生
- **カラオケ効果** — テキスト選択後、読み上げ中に単語が一つずつハイライト
- **即時翻訳** — テキスト選択で Bing または Google 翻訳
- **キーボードショートカット** — 主要機能へのクイックアクセス

---

## 使い方

### ステップ 1: 拡張機能をインストール

[Chrome ウェブストア](https://chromewebstore.google.com/) から **Spanish Reader** をインストールします。

### ステップ 2: ウェブページを開く

スペイン語コンテンツのあるページにアクセスします。

### ステップ 3: 発音を有効化

ツールバーの拡張機能アイコンをクリックし、「発音を有効化」をオンにします。「ページ全体モード」で全単語に注釈を表示できます。

### ステップ 4: 発音を確認

単語にマウスオーバーで発音ツールチップを表示。スピーカーアイコンで発音を再生。音節区切りで単語構造を確認（例：「com·pu·ta·do·ra」）。

### ステップ 5: 読み上げと翻訳

スペイン語テキストを選択するとフローティングツールバーが表示：
- **🔊 読み上げ** — カラオケ風に逐語ハイライト
- **🌐 翻訳** — インライン翻訳バブルを表示

## キーボードショートカット

| ショートカット | Mac | 機能 |
|---------------|-----|------|
| `Alt+Shift+I` | `Ctrl+Shift+I` | 発音注釈のオン/オフ |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 選択テキストを読み上げ |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 選択テキストを翻訳 |

---

## 設定

| 設定 | 説明 |
|------|------|
| **発音を有効化** | マスタースイッチ |
| **ページ全体モード** | 全単語に発音を表示 |
| **アクセント** | ラテンアメリカまたはスペイン |
| **音節区切り表示** | 音節分割マークを表示 |
| **ホバーモード** | 辞書、ツールチップ、オフ |
| **読み上げ速度** | TTS の速度調整 |
| **翻訳エンジン** | Bing または Google 翻訳 |
| **翻訳先言語** | 翻訳のターゲット言語 |
| **PDF スマート検出** | PDF URL を自動リダイレクト |
